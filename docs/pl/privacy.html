OFFLINE XIANGQI

Polityka prywatności

Ostatnia aktualizacja: 2026-06-23

Informacje, które zbieramy

Xiangqi offline nie wymaga konta i nie zbiera aktywnie imienia, telefonu, e-maila, lokalizacji ani kontaktów.

Dane lokalne

Postęp gier, ustawienia i lokalny status okresu próbnego są przechowywane na urządzeniu, aby przywracać gry i zachować preferencje.

Zakupy i przywracanie

Dostęp dożywotni jest obsługiwany przez zakup w aplikacji Apple App Store. Apple przetwarza zakupy, przywracanie i weryfikację uprawnień; aplikacja odczytuje tylko niezbędny status zakupu.

Udostępnianie stronom trzecim

Nie używamy zewnętrznych SDK reklamowych ani analitycznych i nie sprzedajemy, nie wynajmujemy ani nie udostępniamy danych osobowych.

Kontakt

W sprawach prywatności lub pomocy skontaktuj się przez GitHub Issues.
