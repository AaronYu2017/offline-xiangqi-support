离线象棋支持

离线象棋是一款专注于纯粹对弈体验的中国象棋应用。无需联网、无需注册，打开即可开始对局。

支持方式

如果你遇到问题，或希望提交建议，可以通过 GitHub Issues 联系我们。

GitHub Issues 查看隐私政策

常见问题

  • 可以在没有网络时使用吗？
    可以。核心对局功能支持离线使用，也支持飞行模式。购买、恢复购买和打开 App Store 链接时需要使用 Apple 服务。
  • 支持哪些模式？
    支持同屏双人对战和本地 AI 人机对战。
  • 购买后如何恢复？
    在设置页点击“恢复购买”，使用同一个 Apple ID 即可恢复永久解锁。
  • 支持哪些设备和语言？
    支持 iPhone 和 iPad，目前支持 18 种语言。
