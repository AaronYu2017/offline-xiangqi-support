Assistance Xiangqi hors ligne

Xiangqi hors ligne est conçu pour jouer aux échecs chinois simplement. Aucun compte n’est requis et le jeu principal fonctionne sans internet.

Obtenir de l’aide

Pour signaler un problème ou envoyer un retour, contactez-nous via GitHub Issues.

GitHub Issues Politique de confidentialité

FAQ

  • Puis-je jouer sans internet ?
    Oui. Le jeu principal fonctionne hors ligne et en mode Avion. Les achats, restaurations et liens App Store utilisent les services Apple.
  • Quels modes sont disponibles ?
    Mode deux joueurs local et mode IA locale.
  • Comment restaurer un achat ?
    Ouvrez Réglages et touchez Restaurer les achats avec le même identifiant Apple.
  • Quels appareils et langues sont pris en charge ?
    L’app prend en charge iPhone, iPad et 18 langues.
