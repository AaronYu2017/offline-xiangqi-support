OFFLINE XIANGQI

Offline Xiangqi ondersteuning

Offline Xiangqi is gemaakt voor eenvoudig Chinees schaken. Geen account nodig en het hoofdspel werkt zonder internet.

Ondersteuning

Neem contact met ons op via GitHub Issues als je een probleem hebt of feedback wilt geven.

GitHub Issues Privacybeleid

Veelgestelde vragen

  • Kan ik zonder internet spelen?
    Ja. Het hoofdspel werkt offline en in vliegtuigmodus. Aankopen, herstel van aankopen en App Store-links gebruiken Apple-diensten.
  • Welke modi worden ondersteund?
    Lokale modus voor twee spelers en lokale AI-modus.
  • Hoe herstel ik een aankoop?
    Open Instellingen en tik op Herstel aankopen met dezelfde Apple ID.
  • Welke apparaten en talen worden ondersteund?
    De app ondersteunt iPhone, iPad en 18 talen.
