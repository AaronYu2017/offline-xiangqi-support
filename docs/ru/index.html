Поддержка Сянци офлайн

Сянци офлайн создана для простой игры в китайские шахматы. Вход не требуется, а основная игра работает без интернета.

GitHub Issues Политика конфиденциальности

Вопросы и ответы

  • Можно играть без интернета?
    Да. Основная игра работает офлайн и в авиарежиме. Покупки, восстановление покупок и ссылки App Store используют сервисы Apple.
  • Какие режимы поддерживаются?
    Локальный режим для двух игроков и локальная игра против ИИ.
  • Как восстановить покупку?
    Откройте настройки и нажмите «Восстановить покупки», используя тот же Apple ID.
  • Какие устройства и языки поддерживаются?
    Приложение поддерживает iPhone, iPad и 18 языков.
