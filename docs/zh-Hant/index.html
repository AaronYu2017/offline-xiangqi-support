離線象棋支援

離線象棋是一款專注於純粹對弈體驗的中國象棋應用。無需連網、無需註冊，打開即可開始對局。

支援方式

如果你遇到問題，或希望提出建議，可以透過 GitHub Issues 聯絡我們。

GitHub Issues 查看隱私政策

常見問題

  • 可以在沒有網路時使用嗎？
    可以。核心對局功能支援離線使用，也支援飛航模式。購買、恢復購買和開啟 App Store 連結時需要使用 Apple 服務。
  • 支援哪些模式？
    支援同屏雙人對戰和本地 AI 人機對戰。
  • 購買後如何恢復？
    在設定頁點擊「恢復購買」，使用同一個 Apple ID 即可恢復永久解鎖。
  • 支援哪些裝置和語言？
    支援 iPhone 和 iPad，目前支援 18 種語言。
