隱私政策

更新日期：2026-06-23

我們收集的資訊

離線象棋不要求註冊登入，也不會主動收集姓名、手機號碼、電子郵件、位置、通訊錄等個人資訊。

本地資料

對局進度、設定偏好和本地試用狀態僅保存在你的裝置本地，用於恢復對局和保持設定。

購買與恢復購買

永久解鎖透過 Apple App Store 的 App 內購買完成。購買、恢復購買和權益驗證由 Apple 處理，本 App 只讀取必要的購買狀態。

第三方共享

我們不接入第三方廣告或統計 SDK，不會出售、出租或共享你的個人資訊。

聯絡我們

如有隱私或支援相關問題，可以透過 GitHub Issues 聯絡我們。
