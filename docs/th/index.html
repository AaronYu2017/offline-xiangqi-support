หมากรุกจีนออฟไลน์ออกแบบมาเพื่อการเล่นหมากรุกจีนที่เรียบง่าย ไม่ต้องเข้าสู่ระบบ และฟังก์ชันหลักเล่นได้โดยไม่ต้องใช้อินเทอร์เน็ต

การขอความช่วยเหลือ

หากพบปัญหาหรือต้องการส่งคำแนะนำ ติดต่อเราได้ผ่าน GitHub Issues

GitHub Issues นโยบายความเป็นส่วนตัว

คำถามที่พบบ่อย

  • เล่นโดยไม่มีอินเทอร์เน็ตได้ไหม?
    ได้ ฟังก์ชันการเล่นหลักทำงานแบบออฟไลน์และในโหมดเครื่องบิน การซื้อ การกู้คืนการซื้อ และลิงก์ App Store ใช้บริการของ Apple
  • รองรับโหมดใดบ้าง?
    รองรับโหมดผู้เล่นสองคนบนอุปกรณ์เดียวกันและโหมด AI ในเครื่อง
  • กู้คืนการซื้ออย่างไร?
    เปิดการตั้งค่าแล้วแตะกู้คืนการซื้อ โดยใช้ Apple ID เดียวกับที่ซื้อ
  • รองรับอุปกรณ์และภาษาใด?
    รองรับ iPhone, iPad และ 18 ภาษา
