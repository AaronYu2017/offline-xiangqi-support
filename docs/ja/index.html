オフライン象棋 サポート

オフライン象棋は、中国象棋をすぐに遊ぶためのシンプルなアプリです。サインイン不要で、主要な対局機能はインターネットなしで使えます。

サポート

問題の報告やフィードバックは GitHub Issues からご連絡ください。

GitHub Issues プライバシーポリシー

よくある質問

  • インターネットなしで使えますか？
    はい。主要な対局機能はオフラインおよび機内モードで利用できます。購入、購入の復元、App Store リンクは Apple のサービスを使用します。
  • どのモードに対応していますか？
    同じ端末での2人対戦と、ローカル AI 対戦に対応しています。
  • 購入を復元するには？
    設定で「購入を復元」をタップし、購入時と同じ Apple ID を使用してください。
  • 対応デバイスと言語は？
    iPhone、iPad、18言語に対応しています。
