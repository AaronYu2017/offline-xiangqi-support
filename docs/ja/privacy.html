収集する情報

オフライン象棋はアカウントを必要とせず、名前、電話番号、メールアドレス、位置情報、連絡先などの個人情報を積極的に収集しません。

ローカルデータ

対局の進行状況、設定、ローカル試用状態は、対局の復元と設定保持のため端末内に保存されます。

購入と復元

永久解放は Apple App Store のアプリ内課金で処理されます。購入、復元、権利確認は Apple が処理し、アプリは必要な購入状態のみを読み取ります。

第三者との共有

第三者の広告 SDK や分析 SDK は使用しておらず、個人情報を販売、貸与、共有しません。

お問い合わせ

プライバシーまたはサポートに関する質問は GitHub Issues からご連絡ください。
