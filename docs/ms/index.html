OFFLINE XIANGQI

Sokongan Catur Cina Offline

Catur Cina Offline dibina untuk permainan xiangqi yang ringkas. Tiada log masuk diperlukan dan permainan utama berfungsi tanpa internet.

Sokongan

Jika anda menghadapi masalah atau ingin memberi maklum balas, hubungi kami melalui GitHub Issues.

GitHub Issues Dasar Privasi

Soalan lazim

  • Bolehkah saya bermain tanpa internet?
    Ya. Permainan utama berfungsi di luar talian dan dalam Mod Pesawat. Pembelian, pemulihan pembelian dan pautan App Store menggunakan perkhidmatan Apple.
  • Mod apa yang disokong?
    Mod dua pemain setempat dan mod AI setempat.
  • Bagaimana memulihkan pembelian?
    Buka Tetapan dan ketik Pulihkan Pembelian dengan Apple ID yang sama.
  • Peranti dan bahasa apa yang disokong?
    Aplikasi menyokong iPhone, iPad dan 18 bahasa.
