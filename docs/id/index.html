OFFLINE XIANGQI

Dukungan Catur China Offline

Catur China Offline dibuat untuk bermain xiangqi secara sederhana. Tidak perlu masuk akun, dan permainan utama berfungsi tanpa internet.

Dukungan

Jika menemukan masalah atau ingin memberi masukan, hubungi kami melalui GitHub Issues.

GitHub Issues Kebijakan Privasi

Tanya jawab

  • Bisakah dimainkan tanpa internet?
    Bisa. Permainan utama berfungsi offline dan dalam Mode Pesawat. Pembelian, pemulihan pembelian, dan tautan App Store menggunakan layanan Apple.
  • Mode apa yang didukung?
    Mode dua pemain lokal dan mode AI lokal.
  • Bagaimana memulihkan pembelian?
    Buka Pengaturan dan ketuk Pulihkan Pembelian dengan Apple ID yang sama.
  • Perangkat dan bahasa apa yang didukung?
    Aplikasi mendukung iPhone, iPad, dan 18 bahasa.
