오프라인 샹치 지원

오프라인 샹치는 중국 장기 대국에 집중한 간단한 앱입니다. 로그인 없이, 핵심 대국 기능은 인터넷 없이 사용할 수 있습니다.

지원 받기

문제나 의견이 있다면 GitHub Issues를 통해 연락해 주세요.

GitHub Issues 개인정보 처리방침

자주 묻는 질문

  • 인터넷 없이 사용할 수 있나요?
    예. 핵심 대국 기능은 오프라인 및 비행기 모드에서 작동합니다. 구매, 구매 복원, App Store 링크는 Apple 서비스를 사용합니다.
  • 어떤 모드를 지원하나요?
    같은 기기에서의 2인 대국과 로컬 AI 대국을 지원합니다.
  • 구매를 복원하려면 어떻게 하나요?
    설정에서 구매 복원을 누르고 기존 구매에 사용한 Apple ID를 사용하세요.
  • 지원 기기와 언어는 무엇인가요?
    iPhone, iPad 및 18개 언어를 지원합니다.
