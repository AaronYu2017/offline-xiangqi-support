OFFLINE XIANGQI

Soporte de Ajedrez Chino offline

Ajedrez Chino offline está diseñado para jugar xiangqi de forma simple y concentrada. No requiere inicio de sesión y el juego principal funciona sin internet.

Soporte

Si encuentras un problema o quieres enviar comentarios, contáctanos mediante GitHub Issues.

GitHub Issues Política de privacidad

Preguntas frecuentes

  • ¿Puedo jugar sin internet?
    Sí. El juego principal funciona sin conexión y en modo avión. Las compras, restauraciones y enlaces de App Store usan servicios de Apple.
  • ¿Qué modos incluye?
    Modo local de dos jugadores y modo contra IA local.
  • ¿Cómo restauro una compra?
    Abre Ajustes y toca Restaurar compras con el mismo Apple ID usado para comprar.
  • ¿Qué dispositivos e idiomas son compatibles?
    La app es compatible con iPhone, iPad y 18 idiomas.
