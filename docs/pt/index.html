OFFLINE XIANGQI

Suporte do Xadrez Chinês offline

Xadrez Chinês offline foi criado para jogar xiangqi de forma simples. Não exige login e o jogo principal funciona sem internet.

Suporte

Se encontrar um problema ou quiser enviar feedback, fale conosco pelo GitHub Issues.

GitHub Issues Política de Privacidade

Perguntas frequentes

  • Posso jogar sem internet?
    Sim. O jogo principal funciona offline e no Modo Avião. Compras, restauração e links da App Store usam serviços da Apple.
  • Quais modos são suportados?
    Modo local para dois jogadores e modo de IA local.
  • Como restauro uma compra?
    Abra Ajustes e toque em Restaurar Compras usando o mesmo Apple ID.
  • Quais dispositivos e idiomas são compatíveis?
    O app suporta iPhone, iPad e 18 idiomas.
