隐私政策

更新日期：2026-06-23

我们收集的信息

离线象棋不要求注册登录，也不会主动收集姓名、手机号、邮箱、位置、通讯录等个人信息。

本地数据

对局进度、设置偏好和本地试用状态仅保存在你的设备本地，用于恢复对局和保持设置。

购买与恢复购买

永久解锁通过 Apple App Store 的应用内购买完成。购买、恢复购买和权益验证由 Apple 处理，本应用只读取必要的购买状态。

第三方共享

我们不接入第三方广告或统计 SDK，不会出售、出租或共享你的个人信息。

联系我们

如有隐私或支持相关问题，可以通过 GitHub Issues 联系我们。
